改修版ウェブサイト ファイル一式
================================

この度はファイルをご利用いただきありがとうございます。
アップロードいただいたウェブサイトに、英語と日本語の言語切り替え機能を追加したファイル一式です。


## ファイル構成

/ (ルート)
|-- index.html       (メインページ)
|-- blog.html        (活動報告ページ)
|-- css/
|   |-- style.css    (スタイルシート)
|-- js/
|   |-- i18n.js      (言語切り替えと動的処理を行うスクリプト)
|-- locales/
|   |-- en.json      (英語のテキストファイル)
|   |-- ja.json      (日本語のテキストファイル)
|-- images/
|   |-- logo.avif
|   |-- sakura_banner.jpg
|-- README.txt       (このファイル)


## 主な変更点

1.  **言語切り替え機能の追加**
    -   ヘッダーに「日本語 / English」の切り替えスイッチを設置しました。
    -   ユーザーが選択した言語はブラウザに保存され、次回訪問時も同じ言語で表示されます。

2.  **HTMLファイルの修正**
    -   `index.html` および `blog.generated.html` (→`blog.html`に改名) のテキスト部分を、多言語対応のキーに置き換えました。
    -   `<html>` タグのデフォルト言語を `en`（英語）に変更しました。

3.  **JavaScriptの統合と整理**
    -   `index.html` のインラインスクリプトと `main.js` の内容は、新しい `js/i18n.js` に統合・整理しました。
    -   これにより、今後のメンテナンスがしやすくなっています。
    -   **注意**: 元の `main.js` と `config.js` は不要になりました。HTMLからは `js/i18n.js` のみを読み込んでいます。

4.  **動的コンテンツの多言語化**
    -   HSPポイント計算やデジタルノマド適性チェックの結果表示など、動的に生成されるメッセージも言語に応じて切り替わります。
    -   Google Sheetsから取得している「ビザ比較」や「生活サポート」のデータも、言語設定に応じて対応するシート(`..._en` or `..._ja`)から取得するように変更しました。
    -   **重要**: この機能が正しく動作するためには、**Google Sheets側で、現在のシートをコピーして末尾に `_ja` と `_en` をつけたシートをそれぞれ作成し、内容を翻訳していただく必要があります。**
        -   例: `visa_comparison` → `visa_comparison_ja`, `visa_comparison_en`

## 設定方法

1.  上記の手順に従い、フォルダとファイルを作成・配置してください。
2.  お使いのGoogle Sheetsを編集し、上記「4. 動的コンテンツの多言語化」で説明したように、各データシートの日本語版と英語版を作成してください。
